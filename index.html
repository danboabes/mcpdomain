7 MCP Tools

Everything an AI agent needs to own a domain.

Each tool has a detailed agentic description that tells the LLM exactly when to use it. Free tools work without authentication. Registration and configuration require a free MCPDomain account.

01
check_domain_availability Free
Verify whether a specific domain is available. Returns price, TLD detection, and up to 3 alternatives if taken.
02
suggest_available_domains Free
Generate creative domain ideas from keywords or a business description, with real-time availability checks across .com, .io, .ai, and 15+ more TLDs.
03
register_new_domain Auth
Initiate registration. Returns a secure Stripe checkout URL. After payment, the domain is registered and ready in under 60 seconds.
04
configure_domain_email Auth
Set up email forwarding to any inbox (Gmail, Outlook). Catch-all or specific aliases. No MX records required — just works.
05
configure_domain_dns Auth
Add A, CNAME, MX, TXT, AAAA, SRV records. The AI knows the right records for Vercel, Netlify, GitHub Pages, and other hosts.
06
get_my_domain_details Auth
Domain status, DNS records, email config, expiry, and unique AI bot crawling intelligence — which AI bots crawled your domain in the last 30 days.
07
transfer_existing_domain Auth
Move a domain from GoDaddy, Namecheap, or any other registrar to MCPDomain. 1 year added to expiry, all features unlocked.
